# Smart Home Wireless Communication Simulation 
# MSc Dissertation Project
This repository contains MATLAB code developed for evaluating the communication reliability of wireless protocols in smart home security systems.
## Objective 
To compare ZigBee, Wi-Fi, and LoRa based on:
- Packet Loss Ratio (PLR)
- Latency
- Jitter
## Methodology
- Monte Carlo simulation
- Event-driven Poisson traffic model
- Indoor path loss with shadowing
- Retransmission modeling
## Tools Used 
- MATLAB
